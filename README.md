## üìñ Overview

**Shimiyasin** is a responsive, single-page restaurant website designed to evoke a sense of luxury and culinary tradition. It features a deep crimson and gold color palette, glassmorphism UI elements, and interactive AI tools powered by Google Gemini to engage diners.

## ‚ú® Key Features

### üé® UI/UX Design
* **Premium Aesthetic:** "Deep Crimson & Gold" palette with custom typography (Cinzel & Plus Jakarta Sans).
* **Glassmorphism:** Modern, frosted-glass effects on navigation and cards.
* **Micro-interactions:** Smooth hover states, button transforms, and focus rings.
* **Visual Effects:** `tsparticles` integration for a subtle ambient background and AOS (Animate On Scroll) for scroll reveals.

### üçõ Functionality
* **Tabbed Menu:** Filter dishes dynamically by category (All, Seafood, Meat, Desserts).
* **Smart Reservations:** A modal interface that guides users to call the restaurant directly (`+91 77361 07936`).
* **Responsive:** Fully optimized for mobile, tablet, and desktop views.

### ü§ñ AI Kitchen (Powered by Gemini)
* **Dish Historian:** Users can type a dish name (e.g., "Appam") to get a brief history and flavor profile generated by AI.
* **Daily Special Generator:** An AI agent that "scans" seasonal ingredients to invent a unique daily special, complete with a generated description and a simulated visual visualization.

## üöÄ Getting Started

Since this project uses CDN links for libraries, no complex build process (Webpack/Vite) is strictly required.

### Prerequisites
* A modern web browser (Chrome, Firefox, Safari).
* A code editor (VS Code recommended).
* *(Optional)* A Google Cloud API Key for the Gemini features.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/shimiyasin-restaurant.git](https://github.com/yourusername/shimiyasin-restaurant.git)
    ```

2.  **Navigate to the folder:**
    ```bash
    cd shimiyasin-restaurant
    ```

3.  **Open the project:**
    Simply open `index.html` in your browser.
    * *Recommended:* Use "Live Server" extension in VS Code for the best experience.
